## Advent Of Code 2022
###### My repo for solving advent of code in 2022 (https://adventofcode.com/2022)

---

### `BUILD`ing

I'm using bazel. http://bazel.build


Example:

```shell
bazel build //day02:main
```

---

### Running

Example:

```shell
bazel-bin/day02/main < day02/data/test_input.txt
```
